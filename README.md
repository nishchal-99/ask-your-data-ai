# Ask Your Data AI

An AI-powered analytics assistant that allows users to query business data using natural language.

## 🚀 Features

- Natural Language → SQL conversion
- SQLite database integration
- AI-generated queries with error correction
- Interactive Streamlit UI
- Automatic chart generation

## 🧠 How It Works

User input → AI generates SQL → Query runs → Results + Visualization

## 🛠️ Tech Stack

- Python
- SQLite
- Streamlit
- OpenAI API
- Pandas

## 📊 Example Queries

- Top 5 products by sales
- Sales by region
- Monthly sales trends
- Most profitable category

## ⚡ Setup

```bash
git clone <your-repo>
cd ask-your-data-ai
pip install -r requirements.txt
streamlit run app.py
```
